      MeshInfo

      A Reporter object for tracking mesh information, such as the number of elements, nodes, or degrees-of-freedom. The "items" parameter controls the items computed.

      Example Input Syntax

      The following input file snippet demonstrates the use of the MeshInfo object.

      [Reporters/mesh_info]
  [Reporters/mesh_info]
    type = MeshInfo
  []
[]
      (../../../SoftwareDownloads/moose/test/tests/reporters/mesh_info/mesh_info.i)

        C++ Type:ExecFlagEnum

        Options:NONE, INITIAL, LINEAR, NONLINEAR, TIMESTEP_END, TIMESTEP_BEGIN, FINAL, CUSTOM

        Description:The iteration information to output, if nothing is provided everything will be output.
