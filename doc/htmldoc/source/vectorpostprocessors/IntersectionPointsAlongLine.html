      IntersectionPointsAlongLine

      Get the intersection points for all of the elements that are intersected by a line.

      The CSV output consists of the X, Y and Z coordinates of the intersection points between the faces/sides of the elements and the line.

      commentnote

      This object only supports replicated meshes. A distributed version of this object may be created using the ray tracing module.
