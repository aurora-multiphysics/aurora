      PercentChangePostprocessor

      Computes the percent change of a postprocessor value compared to the value at the previous timestep.

      This postprocessor is deprecated. Please use a ChangeOverTimePostprocessor and set the compute_relative_change parameter to true.

      Input Parameters

      • postprocessorThe name of the postprocessor used for exit criterion

        C++ Type:PostprocessorName

        Options:

        Description:The name of the postprocessor used for exit criterion

      Required Parameters

        Default:TIMESTEP_END

        C++ Type:ExecFlagEnum

        Options:NONE, INITIAL, LINEAR, NONLINEAR, TIMESTEP_END, TIMESTEP_BEGIN, FINAL, CUSTOM, TRANSFER, ALWAYS

        Description:The list of flag(s) indicating when this object should be executed, the available options include NONE, INITIAL, LINEAR, NONLINEAR, TIMESTEP_END, TIMESTEP_BEGIN, FINAL, CUSTOM, ALWAYS.
